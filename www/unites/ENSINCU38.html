Unité d'enseignement ENSINCU38:
"Architecture logicielle JEE"

  • Crédits: 3
  • Cours / Tp : 6h / 24h
  • Localisation: Luminy / Saint-Jérome

Description

Résumé:

L'objet du cours est de présenter l'architecture et ledéveloppement d'applications dans le cadre de la plateformeJEE ( Java Enterprise Edition ). Ce cours prend la suite d'unpremier cours de 30 heures sur JEE. Nous étudionsnotamment

  • la maîtrise des Enterprise Java Bean qui sont lescomposantes métier de la plateforme JEE,
  • le couplage entre EJB et base de données relationnellepour la manipulation des données métier ( Java PersistanceAPI ),
  • l'architecture MVC des applications Web avec Spring MVCet Struts,
  • les principes de sécurisation des applications Web
  • les technologies JSF (Java Server Faces) et GWT (GoogleWeb Toolkit) pour la réalisation d'applications Webriches,
  • les techniques de conception et de réalisation des WebServices (SOAP, XML-RPC, WSDL).

Pré-requis :

  • XML / XPATH / XSLT
  • Technologies du Web ((X)HTML, CSS, Javascript)

Parcours

  • Intégration de systèmes logiciels (ISL)
