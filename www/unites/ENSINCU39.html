Unité d'enseignement ENSINCU39:
"Intégration des données"

  • Crédits: 3
  • Cours / Tp : 15h / 15h
  • Localisation: Luminy / Saint-Jérome

Description

Résumé:

Ce cours est structuré en trois parties :

  • Le modèle Relationnel-Objet : La norme SQL3,l’implantation dans le SGBD Oracle
  • XML et les bases de données : Stockage en base,les fonctions SQLXML
  • La correspondance Objet/Relationnel : Le standardJPA, l’implantation Hibernate

Pré-requis :

  • Bonne connaissance des bases de donnéesrelationnelles

Parcours

  • Intégration de systèmes logiciels (ISL)
