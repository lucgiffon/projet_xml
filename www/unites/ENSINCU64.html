Unité d'enseignement ENSINCU64:
"Développement d'applications mobiles"

  • Crédits: 3
  • Localisation: Luminy / Saint-Jérome

Description

Résumé:

Ce cours est dédié au développement d'applications sursmartphones et tablettes dans un environnement Android.L’objectif visé est le développement d’uneapplication complète et professionnelle à l’issue de cecours.

Les éléments suivants seront étudiés :

  • Architecture et OS Android,
  • la conception d’applications (Context,BroadcastReceiver, ContentProviders, …,
  • l’interface graphique (Views, Layouts, MVC,…),
  • le système de ressources et le stockage dedonnées,
  • les librairies,
  • les applications connectées (Client http Android,HttpURLConnection , Google Cloud Messaging,…).

Parcours
